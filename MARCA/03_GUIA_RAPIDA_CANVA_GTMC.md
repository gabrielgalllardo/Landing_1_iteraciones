# Guia Rapida para Canva - GTMC

Objetivo: dejar Canva listo para producir piezas consistentes en minutos.

## 1) Configurar Brand Kit en Canva

1. Ir a Brand -> Brand Kit.
2. Cargar colores con estos HEX:
- #0B0F0E
- #1E3A34
- #A64B34
- #8B3E2B
- #D4735E
- #F5F0E8
- #FAFAF8
- #FFFFFF
3. Definir fuente principal:
- Heading: Space Grotesk
- Subheading: Space Grotesk
- Body: Space Grotesk
4. Crear estilos de texto guardados:
- H1 Hero 64 / Bold / line-height 1.0
- H2 42 / Bold / line-height 1.25
- H3 24 / Bold / line-height 1.25
- Body 16 / Regular / line-height 1.6
- Caption 12 / Semibold / uppercase / tracking alto

Nota: si Canva no ofrece Space Grotesk en tu plan, usar Manrope temporalmente.

## 2) Presets por formato

## 2.1 Post cuadrado (1080x1080)

- Encabezado: 20% alto
- Cuerpo: 55% alto
- CTA: 25% alto
- Colores sugeridos:
1. Fondo #FAFAF8
2. Titular #0B0F0E
3. Acento y boton #A64B34

## 2.2 Historia (1080x1920)

- Encabezado: 18%
- Visual principal: 57%
- Cierre/CTA: 25%
- Colores sugeridos:
1. Fondo #0B0F0E
2. Titular #FFFFFF
3. CTA #A64B34 con texto #FFFFFF

## 2.3 LinkedIn horizontal (1200x627)

- Columna izq (texto): 55%
- Columna der (foto/ilustracion): 45%
- Colores sugeridos:
1. Fondo #F5F0E8
2. Titular #0B0F0E
3. Detalles #A64B34

## 2.4 Documento A4 (presentacion comercial)

- Portada:
1. Fondo #0B0F0E
2. Titulo #FFFFFF
3. Subtitulo #FFFFFF al 70%
4. Linea acento #A64B34
- Paginas internas:
1. Fondo #FFFFFF o #FAFAF8
2. Titulo #0B0F0E
3. Numeros clave #A64B34

## 3) Combinaciones listas por tipo de pieza

1. Educativo:
- Fondo #FAFAF8
- Texto #0B0F0E
- Bullet/iconos #A64B34

2. Testimonio:
- Fondo card #FFFFFF
- Cita #0B0F0E al 60%
- Nombre #0B0F0E
- Marca de cita #A64B34

3. Anuncio CTA:
- Fondo #0B0F0E
- Titular #FFFFFF
- Boton #A64B34
- Boton secundario transparente con borde #FFFFFF al 15%

4. Institucional premium:
- Fondo #1E3A34
- Texto #FFFFFF
- Acento #D4735E (solo titulos o iconos)

## 4) Plantilla de layout reutilizable (copiar y guardar como template)

Bloque A (eyebrow):
- Texto 12, uppercase, #A64B34

Bloque B (titular):
- 1-2 lineas, 42-64, Bold, #0B0F0E o #FFFFFF segun fondo

Bloque C (soporte):
- 2-4 lineas, 16, #0B0F0E al 50-70% o #FFFFFF al 45-70%

Bloque D (accion):
- Boton primario #A64B34 + texto #FFFFFF
- Boton secundario transparente + borde sutil

## 5) Reglas de oro para mantener consistencia

1. No usar mas de 3 colores simultaneos en una pieza.
2. No usar mas de 2 pesos tipograficos fuertes por pantalla.
3. El CTA siempre debe ser el elemento con mas jerarquia visual.
4. Evitar sombras duras o efectos 3D.
5. Usar fotos con overlay oscuro cuando haya texto encima.

## 6) QA rapido antes de exportar

1. Contraste legible en movil.
2. Titulo visible sin zoom.
3. CTA entendible en menos de 3 segundos.
4. Export correcto:
- Redes: PNG
- Impresion: PDF Print (si aplica)
